專案說明

這是我在學習前端程式設計後所完成的第一份大型作業，整體從版面設計、HTML 結構編寫到 CSS 樣式調整，皆由我親自手刻完成，並未依賴 Bootstrap 等前端框架。

為了完成這個專案，我花了不少時間學習與理解 CSS 的排版技術，實作過程中大量運用 Flexbox 與 CSS Grid 來調整區塊排列，並透過 Media Queries 實現響應式設計（RWD），使網站在不同裝置上皆能良好顯示。

在樣式設計上，我採用了語意化的 class 命名方式（如 .container-normal, .nav-dropdown），不僅讓程式碼結構更易讀，也加強了維護性與延展性。

雖然在製作過程中遇到不少挑戰，包括元素對齊、RWD 製作與跨瀏覽器相容性等問題，但透過不斷地測試與調整，我順利完成了這份作品。這次經驗不僅讓我熟悉了 HTML 和 CSS 的實作流程，也讓我體會到手刻網頁雖然繁瑣，卻是建立紮實基礎最有效的方式。

當好不容易做到收尾了突然發現排版亂掉...啊~~~
