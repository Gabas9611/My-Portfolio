專案說明

本專案為一個融合 3D 建模與網頁技術的整合型展示網站，完整涵蓋了以下幾個階段：

3D 模型製作

1.使用 Blender 軟體從頭開始建構模型，包含場景建構、貼圖材質處理與動畫設計。

2.模型經由優化後匯出為 .glb 檔案，方便嵌入網頁並實現即時渲染與互動。

網頁設計與開發

1.使用 HTML、CSS 與 JavaScript 手刻網頁，並輔以 Flexbox、Grid 和 Media Queries 實現響應式設計。

2.採用語義化命名與模組化結構，提升程式碼可讀性與維護性。

功能整合

1.根據指定需求，加入 Loading 動畫頁面、地圖導覽功能、資訊切換與互動按鈕等。

2.模型可透過使用者操作旋轉、縮放，提供沉浸式瀏覽體驗。
